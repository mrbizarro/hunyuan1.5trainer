# HunyuanVideo 1.5 LoRA Trainer

Cog-wrapped trainer for HunyuanVideo 1.5 LoRA fine-tuning on Replicate.

## Features

- HunyuanVideo 1.5 support (Qwen 2.5 VL + ByT5 text encoders)
- Florence-2 auto-captioning for simple image/video uploads
- Memory-optimized training with FP8, block swapping, gradient checkpointing
- Simple ZIP upload - just upload images or videos
